🚀 Sobre Mim

Desenvolvedor Backend em formação, dedicado a criar aplicações robustas e escaláveis com Java e Spring Boot. Possuo uma sólida base em arquitetura de microsserviços, padrões de segurança (JWT/Spring Security ) e práticas de código limpo. Tenho aprimorado minhas habilidades em Amazon Web Services (AWS) para desenvolver e implantar soluções eficientes na nuvem.

Atualmente, estou cursando Análise e Desenvolvimento de Sistemas (ADS) na Unifametro, onde aprofundo meus conhecimentos em engenharia de software, arquitetura de aplicações e boas práticas de desenvolvimento. Durante meus estudos, tenho desenvolvido projetos práticos que aplicam conceitos reais do mercado.

Com uma trajetória iniciada em suporte técnico, desenvolvi uma visão prática voltada à resolução ágil de problemas e compreensão das necessidades do cliente. Busco minha primeira oportunidade como Desenvolvedor Backend Java Júnior para contribuir com times de alta performance e continuar evoluindo tecnicamente.
